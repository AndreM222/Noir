# NOIR

**A Continuity Layer for Modern Cinema**

NOIR is an early-stage system designed to unify fragmented movie consumption across streaming platforms, devices, and environments.

Instead of hosting content, NOIR focuses on coordinating the experience around it—helping users track, resume, and navigate what they watch without losing context.

---

## Vision

Modern movie consumption is fragmented:
- Multiple streaming platforms
- Multiple devices
- Disconnected viewing experiences

NOIR aims to simplify this by acting as a unified layer across platforms.

> A movie should never be lost.

---

## What NOIR Does

NOIR acts as:

- A **movie tracking system**
- A **progress and session manager**
- A **launcher into external streaming platforms**
- A **continuity layer across devices**

It does **not** replace streaming services—it works alongside them.

---

## MVP Features

The current version focuses on a minimal, high-impact feature set:

- Multi-movie tracking (tab-based system)
- Progress tracking across sessions
- Quick resume functionality
- External playback launching

---

## Architecture Overview

The system is designed with:

- **Platform-agnostic integration** (deep linking to services)
- **Local-first state management**
- **Modular feature structure**
- **Non-intrusive interaction model**

---
